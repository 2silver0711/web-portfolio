2026.01 ~ 02

회사를 설명하는 공간에서 방문자의 전시 정보를 찾는 출발점으로 개선

100% / UX 리서치 및 기획 전반, 디자인 작업, 가설 및 KPI 설정

Overview

이 과제에서는 메쎄이상 메인 홈을 '회사를 설명하는 공간'이 아니라 '방문자가 전시 정보를 찾기 시작하는 출발점'으로 리뉴얼하는 것을 목표로 했습니다. 핵심은 단순한 UI 개선이 아니라, 사용자가 수행하려는 태스크를 페이지의 중심에 두는 구조적 전환입니다.

Problem

메쎄이상은 국내 1위 전시 주최사로 연간 다수의 전시를 운영하고 있지만, 현재 메인 홈은 기업 규모와 성과를 설명하는 브랜딩 페이지에 가까운 구조입니다. 반면 실제 방문자의 다수는 관람객과 참가사로, '전시 일정, 참여 방법, 사전 등록' 같은 전시 액션 정보를 찾기 위해 유입됩니다. 즉, 회사가 보여주고 싶은 메시지와 사용자가 수행하려는 태스크가 메인에서 충돌하고 있었고, 이 불일치가 초기 탐색 흐름을 지연시키는 핵심 원인이라고 판단했습니다.

Solution

홈 진입 시 방문자의 '전시 탐색 시작자'로서의 인지 시간을 단축하기 위해, 전시 정보와 회사 정보를 재배치하여 설계했습니다. 핵심 솔루션은 다음과 같습니다.

1. 전시 관람과 참가 버튼을 전시 탐색 CTA로 정의
전시 탐색의 공식적인 시작점을 기존 하단 배너가 아닌 상단 Hero 영역 내 '전시 관람 버튼'과 '참가/참관 버튼'으로 전환하여, 가장 먼저 안내하도록 변경.

2. 전체 일정 버튼 재정의
전체 전시 정보를 확인할 수 있는 '전체 행사일정'을 확인하고 입력한 탐색 흐름의 시작점으로 정의하여 전시 탐색 흐름의 정보 계층을 통합.

3. 전시 정보 즉시 노출
호버링 시 '전시 제목/날짜/장소'를 바로 인지할 수 있도록 함으로써 메인에서부터 전시 정보를 즉시 확인할 수 있게 함.

Result

"전시 탐색 소요시간 약 6~7초에서 3초 이내로 단축" / "전시 탐색 CTA 클릭률 62%에서 83%로, 21%p 상승"(Maze를 활용한 UT 진행 N=6, 관람 목적 4명/참가 목적 2명, 메인 홈 기준 태스크 수행).

중요한 건 수치 자체보다, 사용자가 더 빠르게 '행동을 시작한다'는 변화입니다. 메인은 모든 사용자가 반드시 거치는 핵심 시작점이고, 홈 체제를 다루거나 사용자 행동을 가장 크게 변화시킬 수 있는 지점입니다. '회사를 보여주는 공간'이 아닌 '사용자의 행동을 도와주는 시작점'이라는 가설을 세우고, 그에 맞는 구조를 개선했습니다.
